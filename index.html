SSL Manager

多云平台 SSL 证书自动管理工具，支持自动申请、DNS 验证、续期和部署

🚀 快速开始 ⭐ GitHub

核心功能

📄

自动申请免费证书

支持从阿里云、腾讯云等平台自动申请免费 DV SSL 证书，无需人工干预

🌐

自动 DNS 验证

自动添加 DNS TXT 记录完成域名所有权验证，支持跨云平台验证

🔄

自动续期管理

定时检查证书有效期，在即将过期时自动续期，确保服务不中断

💾

守护进程模式

支持后台持续运行，每 24 小时自动检查证书状态（可配置）

📚

多域名批量管理

通过配置文件统一管理多个域名的证书，支持通配符域名

⚙

后置命令执行

证书更新后自动执行自定义命令，如重载 Nginx、同步到其他服务器

支持平台

☁ 阿里云

  • ✓ 证书申请（DigiCert 免费 DV）
  • ✓ DNS 验证（云解析 DNS）
  • ✓ 证书下载

☁ 腾讯云

  • ✓ 证书申请（TrustAsia 免费 DV）
  • ✓ DNS 验证（DNSPod）
  • ✓ 证书下载

☁ 华为云

  • ✗ 证书申请（API 暂不支持）
  • ✓ DNS 验证
  • ✓ 混合模式支持

快速开始

1 下载

# Linux (amd64) wget https://github.com/your-repo/ssl-manager/releases/latest/download/ssl-manager-linux-amd64 chmod +x ssl-manager-linux-amd64

2 配置

# config.yaml providers: tencent: secret_id: "your-secret-id" secret_key: "your-secret-key" domains: - name: example.com provider: tencent renew_days: 30 post_command: "nginx -s reload"

3 运行

# 单次检查和申请 ./ssl-manager config.yaml # 启动守护进程 ./ssl-manager config.yaml start # 查看状态 ./ssl-manager config.yaml status
